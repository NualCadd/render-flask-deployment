Python is a high-level, interpreted language used for web development, data science, AI, and more.

# Example

Chapter 3: Control Flow & Loops

Use if, for, and while to control program flow.

if x > 0:
    print("Positive")

for i in range(5):
    print(i)

while x > 0:
    x -= 1

Explanation: if statement will check either true or false.
if false => skip.
if true => run the code block inside.
For i in range(x) mean i will run from 0 -> x-1.
While x > 0 mean this loop will run forever until x not > 0.

Chapter 5: File I/O & Exceptions

Read and write files, handle errors gracefully.

with open('data.txt', 'r') as f:
    content = f.read()

    value = int('a')

    print('Not a number')

Explanation: Instead of assign value manually, we can read it from other txt files
try syntax will handle error if int('a') does not make an error then value=int('a') if error then the code will jump into except block;

Chapter 6: Intro to OOP

Classes and objects let you model complex data and behavior.

class Person:
    def __init__(self, name):
        self.name = name

p = Person('Alice')
print(p.name)

Class: blueprint for objects. (Person)
Object: instance of a class. (p)
Attribute: data stored in an object. (p.name)
Constructor: __init__ method to initialize object attributes.

HTML/CSS Course

Chapter 1: HTML Basics

HTML structures the content of webpages.

<h1>Hello World</h1>
<p>This is a paragraph.</p>

Explanation: h1 stand for header 1 -> 6 with size descending.
p stand for paragraph.

Chapter 2: CSS Basics

CSS styles your webpages visually.

h1 { color: red; }
p { font-size: 16px; }

Explanation: we can add styling inside the tags like color, font-size,....

Chapter 3: Layouts (Flex & Grid)

Use flexbox and grid to build responsive layouts.

.container { display: flex; }
.grid { display: grid; grid-template-columns: 1fr 1fr; }

Explanation: Display styling will change how the element display on screen.

Chapter 4: Responsive Design

Use media queries and relative units to adapt to devices.

@media (max-width: 600px) {
  .container { flex-direction: column; }
}

Explanation: @media (max-width:600px) mean if the screen is from 600px or below then .container will be applied flex-direction:column;

Tkinter (Python GUI)

Chapter 1: Intro to Tkinter

Tkinter is Python's standard GUI toolkit.

import tkinter as tk
root = tk.Tk()
root.mainloop()

Explanation: root = tk.Tk() will initialize a Tk window and mainloop() will make the Tk window appear on screen.

Chapter 2: Widgets

Buttons, Labels, Entries are basic widgets.

btn = tk.Button(root, text='Click')
label = tk.Label(root, text='Hello')

Explanation: tk.Button and tk.Label will create a button that attached to root(the window at chapter 1) with text "Click".

Chapter 3: Layout (pack/grid/place)

label.pack()
entry.grid(row=0, column=1)

Explanation: Grid can only place object with row and column, pack and place set object with x and y coordinates
